Definición 2.3.15. Orden de un grupo.

Dado un grupo \(G\) y \(g \in G\text{,}\) se define el orden de \(g\) como el mínimo entero positivo \(n\) tal que \(g^n = e\) (con \(e\) el elemento neutro o unitario). Si tal entero \(n\) no existe, se dice que \(g\) tiene orden infinito. El orden de \(g\) se denota por: \(|g| = n\text{.}\)
