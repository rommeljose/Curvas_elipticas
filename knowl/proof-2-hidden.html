Axiomas que deben cumplir las Leyes de Composición Interna para los campos:

  1. Asociatividad:

    Para todo elemento \(a, b, c\text{;}\) pertenecientes a un conjunto cualquiera K, se deben cumplir las identidades:

    \begin{align} a \oplus (b \oplus c) \amp = (a \oplus b) \oplus c\tag{2.1.2}\\ a \odot (b \odot c) \amp = (a \odot b) \odot c\tag{2.1.3} \end{align}
  2. Conmutatividad:

    Para toda \(a, b\) pertenecientes a un conjunto cualquiera K, se deben cumplir las identidades:

    \begin{gather} a \oplus b = b \oplus a\tag{2.1.4}\\ a \odot b = b \odot a\tag{2.1.5} \end{gather}
  3. Existencia de un elemento neutro:
    1. Existe un único elemento \(e = 0\) en un conjunto cualquiera K, tal que para todo \(a\) elemento de K se verifique la identidad: \(a \oplus e = a\text{.}\)
    2. Existe un elemento \(e = 1\) en un conjunto cualquiera K, diferente de \(0\text{,}\) tal que para todo \(a\) elemento de K se verifique la identidad: \(a \odot e = a\text{.}\)
  4. Existencia de elemento opuesto e inverso:
    1. Para cada \(a\) pertenecientes a un conjunto cualquiera K, existe un elemento \(-a\) de K (que llamaremos \(b\)) 14 , tal que para todo \(a\) perteneciente a K se verifique la identidad: \(a \oplus b = 0\text{.}\)
    2. Para cada \(a \neq 0\) pertenecientes a un conjunto cualquiera K, existe un elemento \(a - 1\) de K (que llamaremos \(b\)) 15 , tal que para todo \(a\) perteneciente a K se verifique la identidad: \(a \odot b = 1\text{.}\)
  5. Distributividad de la multiplicación respecto de la adición:

    Para toda \(a, b, c\text{;}\) pertenecientes a un conjunto cualquiera K, se debe cumplir la identidad:

    \begin{gather} a \odot (b \oplus c) = (a \odot b) \oplus (a \odot c)\tag{2.1.6} \end{gather}
