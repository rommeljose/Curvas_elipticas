Párrafo
  1. Dos elementos del conjunto K siempre pueden ser multiplicados:

    Es decir, dado dos elementos arbitrarios \(a\) y \(b\) del conjunto K, existe otro elemento de K unívocamente determinado, llamado producto y denotado por \(a \odot b.\)

  2. Propiedad asociativa:

    Dado, tres elementos arbitrarios \(a,b,c\) del conjunto K, la identidad: \((a \odot b) \odot c = a \odot (b \odot c)\) es siempre válida.

  3. Existencia del elemento neutro:

    Existe un elementos determinado del conjunto K, que llamaremos identidad \(e = 1\text{,}\) tal que todo elemento \(a\) del conjunto K, satisface la relación \(a \odot e = e \odot a = a\text{.}\)

  4. Existencia de elemento inverso:

    Para todo elementos \(a\) arbitrarios del conjunto K, existe un único elemento \(a^{-1}\) llamado su inverso, que satisface la identidad: \(a\cdot a^{-1} = a^{-1}\cdot a\text{;}\) para todos los elementos del conjunto.

  5. Conmutatividad:

    El orden de operación de los elementos no altera el producto, \(a \odot b = b \odot a\text{.}\)
