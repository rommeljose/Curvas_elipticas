Axioma 2.1.3. De la Adición.

Dos números enteros pueden siempre ser sumados, si satisfacen las siguientes condiciones conocidas como axiomas de las adición.

  1. Dos enteros siempre pueden se sumados:

    Es decir; dado dos enteros arbitrarios \(a\) y \(b\text{,}\) existe otro entero unívocamente determinado, llamado suma y denotado por \(a + b.\)

  2. Propiedad asociativa:

    Dado, tres enteros \(a,b,c\) la identidad: \((a + b) + c = a + (b +c)\) es siempre válida.

  3. Existencia del cero:

    Existe un número determinado, que llamaremos cero, tal que todo entero \(a\) satisface la relación \(a + 0 = a\text{.}\)

  4. Existencia de elemento opuestos:

    Para todo entero \(a\text{,}\) existe otro entero \(-a\) llamado su opuesto.

  5. Conmutatividad:

    El orden de los sumandos no altera la suma, \(a + b = b + a\text{.}\)
