Axiomas que deben cumplir las Leyes de Composición Interna para los grupos aditivos:

  1. Asociatividad:

    Para todo elemento \(a, b, c\text{;}\) pertenecientes a un conjunto cualquiera K, se deben cumplir las identidades:

    \begin{align} a \oplus (b \oplus c) \amp = (a \oplus b) \oplus c.\tag{2.3.1} \end{align}
  2. Existencia de un elemento neutro:

    Existe un elemento único \(e = 0\) en un conjunto cualquiera K, tal que para todo \(a\) elemento de K se verifique la identidad: \(a \oplus e = a\text{.}\)

  3. Existencia de elemento opuesto:

    Para cada \(a\) pertenecientes a un conjunto cualquiera K, existe un elemento \(-a\) de K (que llamaremos \(b\)) 2 , tal que para todo \(a\) perteneciente a K se verifique la identidad: \(a \oplus b = 0\text{.}\)

Nota 2.3.8.

Tener presente que los grupos también pueden ser definidos por una operación binaria o ley de composición "multiplicativa"; en ese caso hablamos de grupos multiplicativos.

Ejemplo del simbolismo matemático moderno.

Solo como referencia comparativa, diremos que un matemático requiere pocos símbolos para comprender y comunicar la definición de grupo en lenguaje formal matemático, como se verifica en la siguiente definición de grupo multiplicativo:

  • \(\exists \; !e\in G: e\odot r=r,\forall \; r\in G\text{.}\)

    Que significa: \(\left(G,\odot \right)\) es un monoide o semigrupo con elemento neutro Tabla 2.3.3; que se puede leer en lenguaje natural: Existe un único elemento de G, tal que, e "producto" r es igual a r, para todo r elemento de G.

  • \(\forall \; r\in G, \exists \; \bar{r} : r\odot \bar{r} = e\text{.}\)

    Que significa: \(\left(G,\odot \right)\) verifica la existencia de elemento simétrico para cada uno de sus elementos. Tabla 2.3.3; que se puede leer en lenguaje natural: Para todo \(r\) elemento de G, existe otro \(\bar{r}\) elemento de G (llamado simétrico de \(r\)), tal que, \(r\) "producto" \(\bar{r}\) es igual a e.
