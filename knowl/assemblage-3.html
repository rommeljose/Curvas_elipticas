Solo como referencia comparativa, diremos que un matemático requiere menos símbolos para comprender y comunicar la definición de grupo en lenguaje formal matemático, como se verifica en la siguiente definición de grupo multiplicativo:

  • \(\exists !e\in G: e\odot r=r,\forall r\in G\text{.}\)

    Que significa: \(\left(G,\odot \right)\) es un monoide o semigrupo con elemento neutro Tabla 2.3.3; que se puede leer en lenguaje natural: Existe un único elemento de G, tal que, e "producto" r es igual a r, para todo r elemento de G.

  • \(\forall r\in G, \exists \bar{r} : r\odot \bar{r} = e\text{.}\)

    Que significa: \(\left(G,\odot \right)\) verifica la existencia de elemento simétrico para cada uno de sus elementos. Tabla 2.3.3; que se puede leer en lenguaje natural: Para todo \(r\) elemento de G, existe otro \(\bar{r}\) elemento de G (llamado simétrico de \(r\)), tal que, \(r\) "producto" \(\bar{r}\) es igual a e.
