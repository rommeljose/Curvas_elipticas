Item 3.
Existencia de un elemento neutro:
  1. Existe un elemento \(0\) en un conjunto cualquiera K, tal que para todo \(a\) elemento de K se verifique la identidad: \(a \oplus 0 = a\text{.}\)
  2. Existe un elemento \(1\) en un conjunto cualquiera K, diferente de \(0\text{,}\) tal que para todo \(a\) elemento de K se verifique la identidad: \(a \odot 1 = a\text{.}\)
